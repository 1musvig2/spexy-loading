# spexy-loading

FiveM roleplay sunucuları için sade ve minimalist bir loading screen.  
QBCore ve QBox ile uyumludur.

## 🔴 Canlı Demo

**[Demo'yu Aç](https://1musvig2.github.io/spexy-loading)**

---

## Yapılandırma

Tüm ayarlar `html/js/config.js` dosyasından yönetilir.

| Alan | Açıklama |
|---|---|
| `server.name` | Sunucu adı |
| `server.cfxId` | Oyuncu sayısı için CFX ID |
| `server.maxPlayers` | Maksimum oyuncu sayısı |
| `background.type` | `color` / `image` / `slideshow` / `video` / `particles` |
| `background.particleCount` | Parçacık arka plan yoğunluğu |
| `colors.background` | Arka plan rengi |
| `colors.loadingBar` | Yükleme çubuğu rengi |
| `music.enabled` | Müziği aç / kapat |
| `music.tracks` | Müzik dosyaları listesi |
| `language` | `"tr"` veya `"en"` |
